# Vulkanos Academy - Plataforma de Cursos Interna

Bem-vindo ao repositório do projeto **Vulkanos Academy**, um Sistema de Gestão de Aprendizagem (LMS) moderno, escalável e de alto desempenho, desenvolvido para atender às necessidades de treinamento interno da nossa empresa.

## 🚀 Tecnologias Utilizadas

*   **Backend:** C# .NET 10
*   **Frontend:** Blazor WebAssembly
*   **Banco de Dados:** PostgreSQL / SQL Server
*   **Autenticação:** JWT, Integração com Google e Apple
*   **Armazenamento:** Azure Blob Storage / AWS S3
